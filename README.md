# random16type

- 16通りある4桁の文字列からランダムに1つ決める
- Choose one randomly from 16 possible 4-digit character strings

## Character set

- I/E
- N/S
- T/F
- J/P

## How to run

```bash
go run .
```

アクセス: http://localhost:8080/

## Docker / docker-compose

ビルド&起動:

```bash
docker compose up --build
```

アクセス: http://localhost:8080/
